SMART HOUSE IOT -

Overview

Smart Home with IoT: Turning Devices On/Off and Real-Time Status Monitoring. This project utilizes the Internet of Things (IoT) to create an automated and interconnected living space. This allows you to control and monitor various devices remotely from a web browser, enhancing convenience, security, and potentially, energy efficiency.

Tool

Arduino
ReactJs
ThreeJS
NodeJs
MongoDB

Information

  • Architecture: Single-Page Application (SPA).
  • Real-time communication with Socket.IO
  • Chatfuel: Device Monitoring and Control.
  • JWT for Authentication.
  • Simulating Arduino with Wokwi.
  • Designing User Interfaces with React Bootstrap.
